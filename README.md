# C# Student Practice Assignments

Welcome to **C# Student Practice Assignments**, a collection of small projects and exercises designed to help students learn and practice Object-Oriented Programming (OOP) concepts in C#. This repository contains examples of classes, objects, encapsulation, methods, and basic console applications.

## Projects Included

### 1. Person Class
- **Description:** Demonstrates a simple `Person` class with public fields `Name` and `Age`.
- **Functionality:** Displays the details of a person using a `Details()` method.

### 2. Vehicle Class
- **Description:** Shows how to create a `Vehicle` class with properties `Brand` and `Color`.
- **Functionality:** Implements actions like `Accelerate()`, `Brake()`, and `Horn()`.

### 3. Account Class
- **Description:** Demonstrates encapsulation with private fields `AccountTitle` and `Balance`.
- **Functionality:** Supports operations like `Deposit()`, `Withdraw()`, and checking available balance.

## Features
- Object-Oriented Programming in C#.
- Use of constructors, methods, and encapsulation.
- Practical examples for beginners to understand classes and objects.
